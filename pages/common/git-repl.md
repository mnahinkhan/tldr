# git repl

> Open a Git shell-like command line.
> Part of `git-extras`.
> More information: <https://github.com/tj/git-extras/blob/master/Commands.md#git-repl>.

- Open a Git shell-like command line:

`git repl`

- Run a git command whilst in REPL:

`{{command}}`

- Run a command (not through Git) whilst in REPL:

`!{{command}}`

- Exit REPL:

`exit|quit`
